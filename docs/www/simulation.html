Simulation

Startet den nexowatt-sim Testbetrieb und ordnet automatisch alle relevanten Datenpunkte im App Center zu, damit die EMS-Logiken ohne reale Hardware getestet werden können.

Simulation-Status

Hinweis: Beim Aktivieren wird die aktuelle App-Center Konfiguration als Backup gespeichert und bei „Deaktivieren“ automatisch wiederhergestellt.

Was wird automatisch gemappt?

  • Grid: gridPointPower → nexowatt-sim.*.grid.power_kw
  • PV: pvPower → nexowatt-sim.*.pv.power_kw
  • Speicher: batteryPower/storageSoc + Storage-Control Setpoint → nexowatt-sim.*.storage.*
  • Tarif: priceCurrent + 24h Preis-Kurve → nexowatt-sim.*.tariff.*
  • EVCS: alle Ladepunkte inkl. limit + enable → nexowatt-sim.*.evcs.*
Tipp: Szenarien lassen sich direkt über die States im nexowatt-sim Adapter steuern (Tarifmodus, PV Wetterfaktor, EVCS Plug/Limit, Speicher SoC usw.).
