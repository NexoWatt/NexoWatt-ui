# Roadmap ab NexoWatt UI 0.8.51

## 0.8.51 – 0-Einspeise Anlagen / Export Guard Field Hardening

- Export Guard Setup-Assistent für 0-Einspeise Anlagen.
- Wechselrichter-/Speicher-Write-Mapping besser validieren.
- Batterie-Ladeleistung als PV-Überschuss-Senke priorisieren.
- WR-Abregelung und Speicherladung gemeinsam diagnostizieren.
- fehlende Write-Datenpunkte klarer blockieren.
- Testmodus / Aktivmodus weiterhin strikt trennen.

## 0.8.52 – Mesh Fairness + Export Guard koppeln

- Mesh-Zielgruppen-Budget an Export-Guard-Überschuss koppeln.
- lokale Verbraucher/Speicher vor WR-Abregelung priorisieren.
- abgeregelte kWh im Wallet genauer dem Mesh-Kontext zuordnen.

## 0.8.53 – Community-/Nachbarschaftsledger

- kWh-Anteile je Mesh-Knoten vorbereiten.
- lokaler Anteil, Remote-Anteil, Netzanteil.
- Betreiberexport für Community-/Nachbarschaftsbetrieb.
