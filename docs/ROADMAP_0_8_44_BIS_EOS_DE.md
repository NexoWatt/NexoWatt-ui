# Roadmap ab NexoWatt UI 0.8.44

## 0.8.44 – Mesh Local Bridge Mapping

Ziel: Die neutralen Command-Intents aus Mesh/Microgrid gezielt lokalen Knoten-Zielgruppen zuordnen.

Geplant:

- Bridge-Mapping je Knoten im Reiter Mesh/Microgrid
- lokaler Ziel-Command-State je Knoten oder Knoten-Gruppe
- Zieltypen: Ladepunkt, Speicher, Verbraucher, Erzeuger, Generic
- Command-Intent-Filter nach Kategorie und Richtung
- Safety-Hinweise je Bridge-Zuordnung
- keine direkten Hardwarewrites aus der Mesh-App
