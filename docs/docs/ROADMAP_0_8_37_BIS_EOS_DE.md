# Roadmap ab NexoWatt UI 0.8.37

## 0.8.37 – Mesh/Microgrid CommandGuard-Vorbereitung

- CommandGuard-Schnittstelle für spätere Microgrid-Aktionen vorbereiten.
- Aktionen weiterhin blockiert lassen, solange kein Freigabemodus aktiv ist.
- Safety-Prüfungen für Netzlimit, Knotenpriorität und Lizenz vorbereiten.
- App-Center-Diagnose für geplante Aktionen erweitern.

## 0.8.38 – Mesh/Microgrid Freigabemodus

- Betreiber kann einzelne geplante Aktionen manuell freigeben.
- Es werden weiterhin nur sichere, neutrale NexoWatt-Intents erzeugt.
- Keine herstellerspezifische Bindung an OCPP, Modbus, MQTT oder REST.

## 0.8.39 – Community-/Nachbarschaftsledger

- Knotenübergreifende lokale kWh-Zuordnung.
- Grundlage für Nachbarversorgung und Energy Sharing.
- Keine eichrechtsverbindliche Abrechnung.

## 0.8.40 – Energy-Hub-/NL-Erweiterung

- Energy-Hub-Kapazitätsbudget.
- NL-spezifische Teruglevering-/Energy-Hub-Diagnose.
- Local First / Grid Last für mehrere Knoten weiter ausbauen.
