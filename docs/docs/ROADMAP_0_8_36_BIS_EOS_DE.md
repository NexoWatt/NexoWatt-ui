# Roadmap ab 0.8.36

## 0.8.36 – Mesh/Microgrid Regelbasis im Diagnosemodus

- Read-only Datenmodell um geplante Entscheidungen erweitern.
- Prioritäten und Netzlimit-Diagnose auswerten.
- Geplante Local-First-/Grid-Last-Aktionen anzeigen.
- Keine Hardware-Schreibbefehle.

## 0.8.37 – Nachbarschafts-/Community-Ledger Vorbereitung

- Ledger-Einträge optional Cluster-/Knotenbezug zuordnen.
- Lokale kWh-Zuordnung für Community/Nachbarschaft vorbereiten.
- Keine Abrechnung, nur Betreiber-/Optimierungsbasis.

## 0.8.38 – Energy Hub / Cluster-Netzlimit

- Netzlimit je Verbund visualisieren.
- Import-/Export-Überlastungen je Cluster anzeigen.
- Prioritäten je Knoten für spätere Steuerung vorbereiten.
