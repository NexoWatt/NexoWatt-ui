# Roadmap ab NexoWatt UI 0.8.60

## 0.8.60 – Explizite Lösch-/Reset-Funktionen

Nach dem Regression-Safety-Gate brauchen bewusste Löschvorgänge eigene, sichere Aktionen:

```text
Speicherfarm wirklich löschen
Ladepunkte wirklich löschen
DC-Stationen wirklich löschen
Mesh-Knoten wirklich löschen
NL/P1-Mapping wirklich löschen
```

Jede Aktion braucht:

```text
Bestätigungsdialog
Vorher/Nachher-Zusammenfassung
Backup-Hinweis
keine automatische Ausführung durch normalen Save
```
